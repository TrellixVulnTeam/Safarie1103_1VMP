Welcome to the Red_Green_Blue Website

The RGB color Model is an additivie color model in which Red,Green and Blue lights are added together to to produce different colors. The model typically uses headecimal numbers 0x0 to 0xF or decimal numbers 0 to 255 for each light to represent the intensity of each respective light. For example the RGB representation of the black color is 0x000 and pure white color is represented by 0xFFF
